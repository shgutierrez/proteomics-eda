# Proteomics EDA (Dataset Fictício)

Este projeto foi desenvolvido para fins **didáticos e de portfólio**, simulando uma análise exploratória de dados de proteômica.  
O dataset fictício contém **7000 entradas**, representando intensidades proteicas em amostras de sangue de adultos masculinos e femininos, coletadas **antes e depois da administração de um fármaco**.

## Objetivos
- Demonstrar técnicas básicas de **Exploração de Dados (EDA)** em Python.  
- Visualizar distribuições de intensidades proteicas.  
- Comparar condições experimentais (antes vs depois do fármaco).  
- Avaliar diferenças entre sexos (masculino vs feminino).  
- Mostrar como combinar código, gráficos e narrativa científica em um notebook.  

## Estrutura do repositório
- `proteomics_dataset.csv` → dataset fictício com 7000 entradas.  
- `proteomics_EDA.ipynb` → notebook Jupyter com análise exploratória.  
- `proteomics_EDA.html` → versão exportada para visualização rápida.  
- `README.md` → documentação do projeto.  

## Tecnologias utilizadas
- Python  
- Pandas  
- Seaborn  
- Matplotlib  

## Observação
Os dados aqui apresentados são **simulados** e não representam resultados reais de experimentos biológicos.  
O foco é **aprender e praticar** análise exploratória, além de ser um guia para quem etá começando.
